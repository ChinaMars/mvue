### Drawer 属性

| 参数              | 说明                          | 类型       | 可选值                         | 默认值   |
|-----------------|-----------------------------|----------|-----------------------------|-------|
| visible         | 是否显示 Drawer，支持 .sync 修饰符 | boolean  | —                           | false |
| position        | Drawer 打开的方向                | position | right / left / top / bottom | right |
| close-by-mask | Drawer是否通过遮罩层关闭             | boolean  | true / false                | true  |
