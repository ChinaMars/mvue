### Select 属性

| 参数          | 说明      | 类型              | 可选值        | 默认值   |
|-------------|---------|-----------------|------------|-------|
| v-model    | 绑定值     | string / number | —          | —     |
| disabled    | 是否禁用    | boolean         | true/false | false |
| scroll-bar | 是否开启滚动条 | boolean         | true/false | false |
| placeholder | 占位符     | string          | —          | 请选择   |

### Select Slots

| slot 名 | 说明          |
|--------|-------------|
| —      | Option 组件列表 |

### Option 属性

| 参数    | 说明    | 类型            | 可选值 | 默认值 |
|-------|-------|---------------|-----|-----|
| value | 选项的值  | string/number | —   | —   |
| label | 选项的标签 | string/number | —   | —   |
