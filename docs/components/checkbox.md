### Checkbox 属性

| 参数       | 说明                                          | 类型                        | 可选值 | 默认值 |
|----------|---------------------------------------------|---------------------------|-----|-----|
| v-model | 绑定值                                         | string / number / boolean | —   | —   |
| label    | 选中状态的值（只有在checkbox-group或者绑定对象类型为array时有效） | string / number / boolean | —   | —   |
| name     | 原生 name 属性                                  | string                    | —   | —   |

### Checkbox-group 属性

| 参数       | 说明                                          | 类型                        | 可选值 | 默认值 |
|----------|---------------------------------------------|---------------------------|-----|-----|
| v-model | 绑定值                                         | array                     | —   | —   |
