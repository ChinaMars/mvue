### 属性

| 参数       | 说明                        | 类型      | 可选值           | 默认值   |
|----------|---------------------------|---------|---------------|-------|
| visible  | 是否显示 Dialog，支持 \.sync 修饰符 | boolean | —             | false |
| title    | Dialog的标题                 | string  | —             | —     |
| position | Dialog的显示位置               | string  | auto / center | auto  |
| width    | Dialog的宽度                 | string  | —             | 30%   |
